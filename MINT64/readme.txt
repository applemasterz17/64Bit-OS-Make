- 실행 
    위 컴파일 된 파일을 ImageMaker 를 통해 Disk.img 파일로 합친다.
    Disk.img 파일은 플로피 디스크용 이미지 파일로써, QEMU 를 통해 부팅 시킨다. 
    QEMU 는 첫번째 섹터(MBR, 0x200 크기) 의 마지막 두 바이트 0x55 0xaa 를 확인하고 부트로더임을 확인한다.
    QEMU BIOS 는 부트로더를 메모리의 0x7c00 영역에 로드한다.
